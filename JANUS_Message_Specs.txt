*** Janus Project ***
** Message Specs **

Message Def
<stx>: 0x02
<length>: 0x1-0xFF length in bytes of the data
<cmd>: 0x1-0xFF Defined in command def
<data>: 
<etx>: 0x03
LRC

Behaviors
-response to valid message is heartbeat, no response to invalid message
-heartbeat response is not heartbeat ( stop endless loop ) 


command def
Server to Node
CA control authority to 
(( Node response to valid message is heartbeat ))
0x01-config
0x02-Login
0x03-CARD ADD
0x04-Clear
0x05-Formats ADD
0x06-Get CARD ID LIST
0x07-HeartBeat
0x08-CARD Remove
0x09-Formats Remove
0x0A-Set Relay Status (On/Off)
0x0B-Set Enabled Status (On/Off) -This allows for machine lockout

Node to Server
0x01-Heart beat ( allows for auto detection )-32bit UTC universal time constant 
0x02-History
0x03-Card Record



[Command]
*Server to Node
0x01-Config
0x02-Login
0x03-CARD ADD: CARD DATA 4-8 bytes
0x04-Clear: 0x1;Clear card, 0x2;Clear config, 0x3;Clear both
0x05-Formats ADD: CARD MASK in hex 4-8 bytes
0x06-Get CARD LIST: ?blank data field?
0x07-HeartBeat: UTC as HEX 4bits wraped in data field
0x08-CARD Remove: CARD DATA 4-8 bytes
0x09-Formats Remove: CARD MASK in hex

*Node to Server
0x01-HeartBeat: UTC as HEX 4bits wraped in data field
0x02-History Record
0x03-Card Record
